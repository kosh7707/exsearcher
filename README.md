# exsearcher

인덱싱 기반 초고속 파일명 검색기. [Everything](https://www.voidtools.com/)을 좋아하지만
**RaiDrive 등으로 마운트된 네트워크 드라이브(NAS)** 까지 색인하고 싶은 사람을 위해 만들었습니다.

- Windows 전용, 포터블 (설치 없음 — 압축 풀고 실행, 폴더 삭제 = 완전 제거)
- 관리자 권한 불필요, 레지스트리/AppData 사용 안 함 (모든 상태가 exe 옆에 저장)
- 색인은 디스크에 저장되어 재시작해도 즉시 검색 가능
- 로컬 드라이브는 실시간 변경 감지, 네트워크 드라이브는 주기 자동 리스캔
- 창을 닫아도 트레이에서 색인을 유지 (종료는 트레이 아이콘 우클릭 → 종료)

## 시작하기

1. zip을 풀고 `exsearcher.exe` 실행
2. 검색창 아래에서 색인할 드라이브 칩을 클릭 — `+ Z: 네트워크` 처럼 `+`가 붙은 칩이 미색인 드라이브
3. 색인이 끝나면 (진행률·예상 시간 표시됨) 바로 검색

드라이브 칩 사용법:

| 동작 | 결과 |
|---|---|
| 미색인 칩(`+ D: 로컬`) 클릭 | 해당 드라이브 색인 시작 |
| 색인된 칩 클릭 | 검색 대상에서 포함/제외 토글 (재색인 아님, 즉시) |
| 칩 우클릭 | 재색인 / 색인에서 제거 |
| `재색인` 버튼 | 색인된 드라이브 전체 재색인 |

## 검색 문법

공백으로 구분한 모든 조건이 **AND**로 묶입니다. 대소문자는 무시합니다.

| 토큰 | 의미 |
|---|---|
| `보고서` | 파일명에 "보고서" 포함 |
| `.pdf` | 파일명이 ".pdf"로 끝남 (확장자) |
| `사진\` | 경로상 상위 폴더 이름에 "사진" 포함 (끝에 `\` 또는 `/`) |

### 유스케이스 예시

| 찾고 싶은 것 | 검색어 |
|---|---|
| 이름에 "계약서" 들어간 모든 파일 | `계약서` |
| 모든 PDF | `.pdf` |
| "사진" 폴더 아래 모든 JPG | `사진\ .jpg` |
| "다운로드" 폴더의 설치 파일 | `다운로드\ setup .exe` |
| 2024가 이름에 붙은 한글 문서 | `2024 .hwp` |
| "백업" 폴더 안의 "프로젝트" 관련 zip | `백업\ 프로젝트 .zip` |
| 특정 드라이브에서만 | 검색어 입력 + 원하는 드라이브 칩만 켜기 |

팁: 폴더 토큰은 경로의 **모든 상위 폴더**와 비교됩니다. `사진\ .jpg`는
`Z:\가족\사진\2024\a.jpg`처럼 중간에 "사진"이 있어도 매치됩니다.

## 결과 사용법

- 더블클릭 / Enter — 파일 열기
- 우클릭 — 열기 / 폴더에서 열기 / 전체 경로 복사
- Esc — 검색어 지우기

## 설정 (settings.ini)

exe 옆 `settings.ini`에 저장됩니다. 주요 키:

| 키 | 기본값 | 의미 |
|---|---|---|
| `index/rescanMinutes` | `30` | 네트워크 드라이브 자동 리스캔 간격(분). `0` = 끔 |
| `index/watchLocal` | `true` | 로컬 드라이브 실시간 변경 감지 |
| `drives/enabled` | (자동) | 검색에 포함할 드라이브 |

리스캔은 백그라운드에서 돌고, 리스캔 중에도 검색은 계속 됩니다.

## Everything과 뭐가 다른가

| | exsearcher | Everything |
|---|---|---|
| NTFS 로컬 드라이브 | 크롤 색인 (수 분) + 실시간 감시 | MFT 직읽기 (수 초) |
| 네트워크 드라이브 (RaiDrive/SMB 마운트) | **색인 지원 + 주기 리스캔** | 미지원(ETP 서버 필요) |
| 관리자 권한 | 불필요 | 필요(서비스 또는 승격) |
| 검색 문법 | 단순 3종 (이름/확장자/폴더) | 매우 풍부 |

로컬 드라이브 위주라면 Everything이 더 빠릅니다. NAS 마운트 드라이브가 주력이면 exsearcher가 답입니다.

## 빌드

```
요구: Visual Studio 2022+ (MSVC), CMake 3.21+, Qt 6.x (msvc 64-bit)

cmake -B build -DCMAKE_PREFIX_PATH=C:/Qt/6.11.1/msvc2022_64
cmake --build build --config Release          # 빌드
cmake --build build --config Release --target deploy   # build/dist/exsearcher/ 에 포터블 폴더 생성
```

코어(색인/검색)는 Qt 비의존 C++20 라이브러리이며 `exsearcher-cli`로 GUI 없이 테스트할 수 있습니다.

## 라이선스 메모
